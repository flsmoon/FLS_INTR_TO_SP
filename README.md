# Трекер привычек

## Описание проекта
Трекер привычек — это простое веб-приложение на Python, которое помогает пользователю создавать полезные привычки и отмечать их выполнение.

Приложение можно использовать для отслеживания ежедневных действий, например:
- пить воду каждый день;
- читать 30 минут;
- делать зарядку;
- учить английский;
- ложиться спать вовремя;
- выполнять домашние задания.

Главная идея проекта — дать пользователю простой инструмент, с помощью которого он сможет видеть свои привычки, отмечать их выполнение и следить за небольшим прогрессом, а также наблюдать «рост цветка» по мере того, как привычка выполняется день за днём.

## Основные функции приложения

1. **Просмотр списка привычек.**
   На главной странице пользователь видит список всех добавленных привычек. Сначала показываются те, которые ещё не выполнены сегодня.

2. **Добавление новой привычки.**
   Пользователь может добавить новую привычку через простую форму. Например: «Читать 20 минут каждый день».

3. **Удаление привычки.**
   Если привычка больше не нужна, пользователь может удалить её из списка. Перед удалением показывается окно с подтверждением.

4. **Отметка привычки как выполненной за сегодняшний день.**
   У каждой привычки есть кнопка «Выполнено». Повторно засчитать выполнение в тот же день нельзя.

5. **Подсчёт количества выполнений.**
   Приложение считает, сколько раз каждая привычка была выполнена за всё время.

6. **Подсчёт количества дней подряд (streak).**
   Если привычка выполнена сегодня, а до этого выполнялась без пропусков — счётчик растёт. Если день был пропущен, streak сбрасывается.

7. **Визуализация прогресса в виде цветка.**
   Для каждой привычки показывается простой emoji-цветок, который «растёт» в зависимости от длины streak:
   - 0 дней — 🌰 семечко;
   - 1–2 дня — 🌱 росток;
   - 3–5 дней — 🌿 листочки;
   - 6–10 дней — 🌸 цветок;
   - 11+ дней — 🌻 большой цветок.

8. **Страница статистики.**
   Показывает общую сводку (всего привычек, всего выполнений, лучший streak) и таблицу по каждой привычке.

## Используемые технологии

- **Python** — основной язык программирования.
- **Flask** — веб-фреймворк.
- **SQLite** — встраиваемая база данных.
- **Jinja2** — шаблоны HTML,встроены во Flask.
- **HTML / CSS** — внешний вид страниц.
- **JavaScript** — только подтверждение удаления через `confirm()`.
- **Git** — контроль версий.

## Схема базы данных

**Таблица `habits`** — список привычек:
| Поле | Тип | Описание |
|------|-----|----------|
| `id` | INTEGER PRIMARY KEY AUTOINCREMENT | Уникальный идентификатор |
| `title` | TEXT NOT NULL | Название привычки |
| `created_at` | TEXT NOT NULL | Дата создания (`YYYY-MM-DD`) |

**Таблица `completions`** — отметки о выполнении:
| Поле | Тип | Описание |
|------|-----|----------|
| `id` | INTEGER PRIMARY KEY AUTOINCREMENT | Уникальный идентификатор |
| `habit_id` | INTEGER NOT NULL | Ссылка на `habits.id` |
| `completed_date` | TEXT NOT NULL | Дата выполнения (`YYYY-MM-DD`) |

## Маршруты приложения

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/` | Главная — список привычек |
| GET | `/add` | Форма добавления привычки |
| POST | `/add` | Сохранить новую привычку |
| POST | `/complete/<habit_id>` | Отметить привычку выполненной за сегодня |
| POST | `/delete/<habit_id>` | Удалить привычку |
| GET | `/stats` | Страница статистики |

## Запуск проекта

### 1. Клонировать репозиторий и перейти в папку

```bash
git clone <url-репозитория>
cd FLS_INTR_TO_SP
```

### 2. Создать виртуальное окружение и установить зависимости

**Windows (PowerShell или cmd):**
```powershell
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

**Linux / macOS:**
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 3. Запустить приложение

```bash
python app.py
```

После запуска в консоли появится сообщение примерно такого вида:

```
 * Running on http://127.0.0.1:5000
```

Откройте в браузере адрес: **http://127.0.0.1:5000**

База данных `habits.db` будет создана автоматически при первом запуске на основе файла `database/schema.sql`.

### 4. Остановить приложение

Нажмите `Ctrl + C` в терминале.
