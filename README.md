# Трекер привычек


## Описание проекта
Трекер привычек — это простое веб-приложение на Python, которое помогает пользователю создавать полезные привычки и отмечать их выполнение.

Приложение можно использовать для отслеживания ежедневных действий, например:
- пить воду каждый день;
- читать 30 минут;
- делать зарядку;
- учить английскиq;
- ложиться спать вовремя;
- выполнять домашние задания.

Главная идея проекта — дать пользователю простой инструмент, с помощью которого он сможет видеть свои привычки, отмечать их выполнение и следить за небольшим прогрессом.
## Основные функции приложения

1. **Просмотр списка привычек**

   На главной странице пользователь видит список добавленных привычек.

2. **Добавление новой привычки**

   Пользователь может добавить новую привычку через простую форму. Например: «Читать 20 минут каждый день».

3. **Удаление привычки**

   Если привычка больше не нужна, пользователь может удалить её из списка.

4. **Отметка привычки как выполненной за сегодняшний день**

   Пользователь может нажать кнопку «Выполнено», чтобы отметить выполнение привычки за текущий день.

5. **Подсчёт количества выполнений**

   Приложение считает, сколько раз каждая привычка была выполнена.

## Предполагаемая архитектура проекта

Проект имеет простую архитектуру.
Он состоит из четырёх основных частей:

### 1. Python-приложение

Главный файл проекта — app.py.
В этом файле находится Flask-приложение и основные маршруты.
Файл app.py отвечает за:

- открытие главной страницы;
- добавление привычки;
- удаление привычки;
- отметку выполнения;
- вывод статистики;
- взаимодействие с базой данных.

### 2. HTML-шаблоны

Папка templates/ содержит HTML-страницы приложения.

Flask использует шаблоны, чтобы подставлять данные из базы данных и показывать их пользователю.
### 3. Статические файлы

Папка static/ содержит файлы оформления.

В первую очередь это CSS-файл, который отвечает за внешний вид приложения.

### 4. База данных SQLite

SQLite используется для хранения данных приложения.
База данных хранится в одном файле, например habits.db.


## Используемые технологии

В проекте предполагается использовать следующие технологии:

- Python — основной язык программирования проекта.
- Flask — простой веб-фреймворк для создания сайта на Python.
- SQLite — лёгкая база данных.
- HTML — используется для создания структуры страниц.
- CSS — используется для оформления интерфейса.
- Git — используется для контроля версий проекта.
